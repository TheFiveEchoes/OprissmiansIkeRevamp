# OprissmiansIkeRevamp
The Repository for the OprissmiansIkeRevamp Planet Pack

# Manual Installation
Step 1: Uninstall any previous installment of OprissmiansIkeRevamp if existing
Step 2: Copy the "OprissmiansIkeRevamp" folder from the downloaded zip into your KSP installation's GameData folder
